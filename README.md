## git 提交初始化命令
**初始化**
```bash
- echo "# demo" >> README.md
- git init
- git add README.md
- git commit -m "first commit"
- git branch -M main
- git remote add origin https://github.com/wangrongguo/data.git
- git push -u origin main
```
**二次提交**
```bash
- git add *
- git commit -m "描述"
- git remote add origin https://github.com/wangrongguo/data.git
- git branch -M main
- git push -u origin main
```
